# ASCII-Birthday v1.0.0

First stable Windows release.

## Included

- Standalone `ASCIIBirthdayGenerator.exe`
- Personalized `[name]_[age]_Birthday.exe` output
- Terminal-style ASCII birthday window
- Animated ASCII confetti
- Ordinal age display such as `21st`, `22nd`, and `25th`
- Any-key close behavior

## Requirements

- Windows 10 or newer
- No Python installation required
